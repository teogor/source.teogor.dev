ContrastThreshold

enum ContrastThreshold : Enum<ContrastThreshold>

Represents the threshold values for contrast ratio compliance.

These thresholds are based on WCAG guidelines and are used to evaluate the contrast ratio between colors to ensure accessibility.

For more information, refer to WCAG Guidelines.
