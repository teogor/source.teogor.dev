MaterialDividerItemDecoration

class MaterialDividerItemDecoration(context: Context, attrs: AttributeSet?, defStyleAttr: Int = R.attr.materialDividerStyle, orientation: Int) : RecyclerView.ItemDecoration

MaterialDividerItemDecoration is a ItemDecoration, similar to a [ ], that can be used as a divider between items of a LinearLayoutManager. It supports both .HORIZONTAL and .VERTICAL orientations. 

dividerItemDecoration = new MaterialDividerItemDecoration(recyclerView.getContext(),
layoutManager.getOrientation());
recyclerView.addItemDecoration(dividerItemDecoration);
