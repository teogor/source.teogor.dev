setTrackDecorationTintList

fun setTrackDecorationTintList(tintList: ColorStateList?)

Applies a tint to the track decoration drawable. Does not modify the current tint mode, which is Mode.SRC_IN by default.

Subsequent calls to .setTrackDecorationDrawable will automatically mutate the drawable and apply the specified tint and tint mode using DrawableCompat.setTintList.

Parameters

tintList

the tint to apply, may be null to clear tint
