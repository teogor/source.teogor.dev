XenoglotLocale

class XenoglotLocale(val language: Language, val country: Country?)

Represents a locale with a specific language and optional country.

This class provides methods to retrieve locale-specific information such as display names and language tags. It also allows setting and getting default locales for different categories.

Parameters

language

The Language associated with this locale.

country

The optional Country associated with this locale.
